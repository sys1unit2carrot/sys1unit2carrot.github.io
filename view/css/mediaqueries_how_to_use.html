CSSでレスポンシブ対応
レスポンシブ対応とは

レスポンシブデザインとは、スマートフォン・タブレット・PCなど、異なる画面サイズの幅を基準にしてWebサイト表示を柔軟に調整し、見やすくすることです。

スマホで表示する用でHTMLを別途用意せずとも、メディアクエリを使用すれば画面の幅ごとにCSSを設定することが可能です。

メディアクエリとは

メディアクエリとは、ブラウザの表示領域やデバイスの画面サイズなどの条件によって適応させるスタイルを切り替えられる機能のことです。

例えば、「画面の横幅が600px以下の時はこんなデザイン！」という指定が可能になります。

使い方①　CSS切り替え

HTMLソースのheadタグ内で画面サイズを指定し、CSSシートを切り替える方法です。

使い方②　CSSに記述

CSS内で画面サイズの指定とプロパティの指定をしてデザインを切り替える方法です。「一部のデザインだけ切り替えたい！」というときに便利です。

Max Width

画面サイズの横幅が指定したサイズより小さい場合に、スタイルが適用されます。

/* 画面サイズの横幅が 768px より小さい場合に適用 */

@media screen and (max-width: 768px) {

　.sample {

　　font-size :10px;

　}

}

Min Width

画面サイズの横幅が指定したサイズより大きい場合に、スタイルが適用されます。

/* 画面サイズの横幅が 768px より大きい場合に適用 */

@media screen and (min-width: 768px) {

　.sample {

　　font-size :10px;

　}

}

複数のWidth指定

画面サイズの横幅が○○以上～○○以下という場合にスタイルが適用されます。

/* 画面サイズの横幅が 480px より大きく、768より小さい場合に適用 */

@media screen and (min-width: 600px) and (max-width: 768px) {

　.sample {

　　font-size :10px;

　}

}
