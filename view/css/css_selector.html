CSSセレクタ
CSSセレクタとは

CSS でスタイルを定義するときに、適用させたい要素を指定することができます。
これに用いられるのが CSS セレクタで、CSS セレクタにはいくつかの書き方があるので紹介します。

CSSセレクタの種類

全称セレクタ(ユニバーサルセレクタ)

全ての要素を対象とする。
「*」を記述。「*」は CSS ではすべての要素という意味を持つ。
HTML で全ての要素にかかる style なので初期状態として使える。
セレクタの中で一番優先度が低いので他のセレクタなどで簡単に打ち消せる。

* {
  color: red;
}

要素型セレクタ(タイプセレクタ)

指定した要素にかかる style。一般的によく見るやつ。

text {
  color: red;
}

IDセレクタ

ID の指定があるときに使用される style。「#」を先頭に付けると ID セレクタになる。

#text {
color: red;
}


赤で表示される

HTML 記述時は「#」を付けない。

クラスセレクタ

クラスの指定があるときに使用される style。「.」を先頭に付けると ID セレクタになる。

.text {
color: red;
}


赤で表示される

HTML 記述時は「.」を付けない。
また、「.」の前に要素を指定することもでき、

h1.text {
  color: red;
}


赤で表示される


赤で表示されない

のようにすることもできる。

属性セレクタ

HTML で要素にセットで記述する id や class などを属性という。
属性セレクタは要素が特定の属性を持つときだけ対象に適用。

h1[id] {
  color: red;
}


赤で表示される


赤で表示されない

セレクタの指定なしで属性のみ指定することも可能。

[id] {
  color: red;
}

属性を複数指定することも可能で属性の値まで指定することも可能。

  h1[id="text"][class] {
    color: red;
  }

属性の値が複数ある場合、指定した属性の値がどれかと合致していたら適用。

p[class~="p3"]{
  color: red;
}


赤で表示される


赤で表示されない

属性の値が指定した文字から始まる場合に適用。

p[name^="abc"] {
  color: red;
}


赤で表示される

属性の値が指定した文字で終わる場合に適用。

p[name$="123"] {
  color: red;
}


赤で表示される

属性の値に指定した文字が含まれている場合に適用。

p[name＊="c1"] {
  color: red;
}


赤で表示される

疑似クラス

カーソルが重なった時などに要素全体に対して処理が行われる。
例 hover, active, linkなど

疑似要素

要素全体ではなく要素の先頭のみなど特定の位置を対象としたいときに使用。
例 before, after, first-letter, first-lineなど

グループ化セレクタ

複数の要素に同じ style を適用させたいときに使用。3つ以上でも可能。

h1, p {
  color: red;
}

CSSセレクタの優先順位

style を定義するとき同じプロパティを2回記述した場合、後に書いた方が優先される。

text {
  color: red;
  color: blue;
}

この場合は blue になる。

同じセレクタ名で2つ定義したとき、後に書いた方が優先される。

  text {
    color: red;
  }
  
  text {
    color: blue;
  }

この場合は blue になる。

例えば下のような状態があったとする。

p {
  color: red;
}

#text {
  color: blue;
}


ここの色は？

この場合、「ここの色は？」という文字は青になる。

CSS セレクタには適用される優先順位があり、この場合は ID セレクタが優先されている。
全体の優先順位を数値化すると、

全称セレクタ0
要素型セレクタ1
疑似要素1
疑似クラス10
クラスセレクタ10
IDセレクタ100
要素に直書き1000

となり、適用されている style の中で一番数値の大きいものが優先される。

.text {
  color: red;
}
.text p {
  color: blue;
}

この場合は、.text は10、 .text p は10と1で11、なので blue が優先される。
点数が同じ場合はあとに書いた方が優先される。

!important 宣言

紹介したように CSS セレクタには優先順位が存在するが、これは必ず適用させたいというものは !important を記述することで最優先で適用させることができる。

p {
  color: red !important;
}

注意点として、簡単に優先度を上げられるが別 style が適用されなくなるので事故を起こしやすかったり構造がわかりにくくなることもあるので極力使用せず CSS セレクタの優先度に従って書く方が安全。
