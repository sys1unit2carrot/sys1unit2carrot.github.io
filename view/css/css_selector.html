CSSのセレクタ
CSSセレクタとは

CSSでスタイルを定義するときに、適用したい要素を指定することができます。
これに用いられるのがCSSセレクタで、CSSセレクタにはいくつかの書き方があるので紹介します。
※テストです。時間があるときに続きを書きます。

CSSセレクタの種類

全称セレクタ(ユニバーサルセレクタ)

全ての要素を対象とする。
「*」を記述。「*」はCSSではすべての要素という意味を持つ。
HTMLで全ての要素にかかるstyleなので初期状態として使える。
セレクタの中で一番優先度が低いので他のセレクタなどで簡単に打ち消せる。

* {
  color: red;
}

要素型セレクタ(タイプセレクタ)

指定した要素にかかるstyle。一般的によく見るやつ。

text {
  color: red;
}
