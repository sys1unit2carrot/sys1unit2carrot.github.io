可変する文字サイズ
文字の大きさを可変させる

CSSプロパティのfont-sizeで値を指定すると、文字のサイズを変更することができます。font-sizeの値はpx以外に指定できる単位があり、画面サイズに比例する単位で指定すると文字の大きさを動的に可変させることができます。

指定できる単位

vw

相対指定。ビューポートの幅の%で決まる。

.a {
  font-size: 3vw;
}

vh

相対指定。ビューポートの高さの%で決まる。

.a {
  font-size: 3vh;
}

vmin

相対指定。ビューポートの「幅か高さの短い方」の%で決まる。

.a {
  font-size: 3vmin;
}

vmax

相対指定。ビューポートの「幅か高さの長い方」の%で決まる。

.a {
  font-size: 3vmax;
}
clampメソッド

CSSの関数で、最小値・基準値・最大値を設定して可変させることができます。

【書き方】
font-size: clamp(最小値, 基準値, 最大値);
【例】
font-size: clamp(15px, 1vw, 30px);

この例だと、ウィンドウ幅に対して1vwの大きさを基本としウィンドウ幅を縮小した場合、15px以下になることはなく、逆にウィンドウ幅を拡大した場合に30px以上になりません。つまり15～30pxの間で可変をしてくれるようになります。
