marginとpadding
marginとpaddingとは

marginとpaddingは余白を調節するために使うCSSプロパティです。
設定する値で余白の大きさが変わります。

marginとpaddingの違い

テキストや画像などを表示する領域の周りにはborderという境界線があります。
borderの内側の余白がpadding、外側の余白がmarginです。

プロパティ指定方法

marginもpaddingも上下左右の値を個別に設定できます。

1. 個別設定

marginの上、paddingの左など、余白の種類(プロパティ)と方向をあらかじめ指定する方法です。「marginの下だけ設定したい！」「paddingの左だけ設定したい！」という場合にオススメです。

/* 個別設定の例 */

.a {

　margin-top: 30px;

　padding-left: 10px;

}

↓　反映させると……

marginがオレンジ、paddingが緑、表示領域が青で表示されています。
marginの上とpaddingの左に余白ができました。

2. 一括設定

上下左右の値をまとめて指定する方法です。
実装後の修正も楽なので、余白を複数設定する場合はこちらの方法がオススメです。値は基本的に「上・右・下・左」の順で指定しますが、「上下左右」「上下・左右」「上・左右・下」で設定することもできます。

/* 一括設定の例 */

.a {

　margin: 10px 5px;

　padding: 5px 10px 30px 20px;

}

↓　反映させると……

marginがオレンジ、paddingが緑、表示領域が青で表示されています。
marginとpaddingの全方向に余白ができました。
